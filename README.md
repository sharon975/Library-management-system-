
# Library Management System

This is my final semester project for database management.  
The system uses SQL to manage books, members, and borrow records.  

## Tables
1. **Books** – stores book details  
2. **Members** – stores member details  
3. **BorrowRecords** – tracks which member borrowed which book  

Each table has 5 sample rows of data.
